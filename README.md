#libec

Libec is a small PKI library leveraging [libsodium](https://github.com/jedisct1/libsodium).

##Installation
```bash
$ git clone https://github.com/erayd/libec.git
$ cd libec
$ ./autogen.sh
$ ./configure
$ make check
# make install
```
